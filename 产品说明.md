# 智能座舱音乐推荐系统

## 产品体验

三层智能架构，让座舱"读懂"此刻的你：

| 层级 | 能力 | 作用 |
|------|------|------|
| 感知层 | 多模态信号采集 | 看见环境、听见氛围、感知状态 |
| 语义层 | 大模型实时推理 | 理解"现在发生了什么" |
| 执行层 | 个性化内容生成 | 决定"该给你什么体验" |

从感知到执行，端到端生成专属体验——座舱不再是冰冷的交通工具，而是懂你的移动空间。

---

## 场景推理举例

| 信号输入 | 智能推理 | 个性化输出 |
|----------|----------|------------|
| 🌙 深夜 + 独自驾驶 | "下班回家的疲惫时刻" | 调取你的「深夜歌单」，舒缓节奏配暖色灯光 |
| 😴 检测疲劳信号 | "需要提神" | 高能电子乐 + 动态音效 + 冷色灯光刺激 |
| 👨‍👩‍👧‍👦 多人 + 儿童乘客 | "家庭出行场景" | 过滤成人内容，欢快儿歌 + 柔和氛围灯 |
| 🏖️ 车外摄像头：海岸线 + 阳光 | "度假模式开启" | 轻快民谣 + 海蓝色灯光 + 低音增强音效 |
| 💑 情侣 + 夜晚 + 城市灯光 | "浪漫约会时刻" | 情歌对唱 + 玫瑰色氛围灯 + 人声增强音效 |

## 核心差异点

- ❌ 不是预设模板匹配
- ✅ 是实时感知 → 语义理解 → 动态生成的端到端智能
