## 核心检查项 Checklist

本检查项用于在项目各阶段进行快速核对，确保核心功能与体验指标达标。

### 1. 接口与数据准备
- [ ] Scene Descriptor JSON Schema (V2.0) 已冻结并分发
- [ ] Feedback Report 结构已冻结并分发
- [ ] ACK 消息结构已冻结并分发
- [ ] Mock 数据中心已搭建 (包含 3-5 个典型用户画像及记忆数据)
- [ ] Event Bus Mocker 工具已就绪

### 2. 核心链路与响应速度
- [ ] Layer 1 & Layer 2 快通道已跑通
- [ ] **体验指标 E1**：上车/场景突变 2 秒内有氛围响应 (快通道)
- [ ] Query Router 与 ACK 旁路已跑通
- [ ] **体验指标 E8**：用户主动说话后 1 秒内收到语音确认 (ACK)
- [ ] Layer 3 慢通道与大模型已接入
- [ ] **体验指标 E2**：慢通道 5-15 秒内完成精细化推理并平滑增量更新

### 3. 引擎集成与协作
- [ ] 编排协调器与各引擎 (内容、灯光、音效) 已集成
- [ ] Event Bus 引擎间通信已打通 (如灯光跟随节拍)
- [ ] 引擎自主决策逻辑已完善 (处理 Hints 采纳与硬约束)
- [ ] Feedback Report 生成与回传机制已跑通

### 4. 体验打磨与质量评估
- [ ] 记忆系统 (短、中、长期) 已实现并能影响推荐结果
- [ ] 自动化规则校验闭环已建立 (100% 拦截不合规输出)
- [ ] 用户行为反馈收集与分析闭环已建立
- [ ] 场景过渡策略 (渐变、事件、紧急) 已优化，过渡自然
- [ ] 场景模板已扩充至 100 个
